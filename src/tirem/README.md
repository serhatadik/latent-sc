The code is written in Anaconda/Pycharm. 

Required Libraries:
numpy
pandas
scipy.io
time
numba
ctypes
math
datetime
matplotlib.pyplot

To obtain TIREM RSSI values and generate features, open main_tirem_pred.py, 
choose the input parameters for the tirem_pred function in the main function, and run.
It will save the relevant files in the current folder and plot the resulting RSSI values,
and generated features.
